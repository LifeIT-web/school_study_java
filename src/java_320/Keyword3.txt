. 단항 산술 연산자

	연산자는 많이 쓸 수록 좋지 않다.

연산자			사용법				의미

+				+A					양수 부호


-				-A					음수 부호


++				++A, A++			증감 연산


--				--A, A--				감소 연산


대입 연산자(Assignment Operator)

: 값이나 식을 변수에 대입하는 용도로 사용함.
   = 은 가장 많이 사용하는 대입 연산자임.
  대입 연산자는 우연에 있는 값을 좌변에 대입.

  . 단순 대입 연산자 :  = 

  . 혼합 대입 연산자 : 다른 연산자와 대입 연산자(=)를 
                                      결합해서 연산을 수행한 다음 좌변에 대입



연산자				사용법					의미

 =					a = b					a = b
 +=					a += b					a = a + b 
 -=					a -=	b					a = a - b
 *=					a *= b					a = a * b
 /=					a /= b					a = a / b
 %=					a %= b					a = a % b
 &=					a &= b					a = a & b
 |=					a |= b					a = a | b
 ^=					a ^= b					a = a ^ b
 <<=				a <<= b					a = a << b
 >>=				a  >>= b					a = a >> b
 >>>=				a >>>= b				a = a >>> b

. 관계 연산자(Relational Operator)

  : 관계연산자는 이항 연산자로 피연산자 사이의 크기를
    비교함.   결과는 언제나 true 아니면 false 이다.
    관계 연산자는 if, while, for 문과 같은 제어문에서 주로 사용


    연산자	

    >
    >=
    <
    <=
    ==
    !=



.논리 연산자(Logical Operator)

: 논리 연산자는 두개의 논리 값을
  평가하며true 또는  false 결과를 반환


연산자			사용법					의미

&				A & B					A와 B 모두 true 일때 true					

&&				A && B					(Short circuit Logical Operator)

|				A | B					A와 B 중 하나 이상이 true

||				A || B

!				!A						A 가 true면 false
										A 가 false면 true





. 비트 연산자(Bitwise Operator)

: 자바는 데이터를 비트단위로 처리할 수 있는
  비트 연산자를 제공합니다. 비트 연산자는
  char, byte. short, int, long 자료형에서만 사용
  

  연산자			연산식				예				의미

  &				a & b				10 & 8			비트단위 and 연산				
  |				a | b					10 | 8			비트단위 or 연산
  ~				~a					~10				비트단위 not 연산
  ^				a ^ b				10 ^ 8			비트단의 xor 연산
  >>				a >> 1				8 >> 1			오른쪽으로 비트단위 이동
  >>>			a >>> 1				8 >>> 1			오른쪽으로 비트단위 이동
													(부호에 관계없이 0이 들어감)
  <<				a  << 1				16 << 1			왼쪽으로 비트단위 이동

  음수면 1 , 양수면 0
