  . 제어문(Control Statement)

  : 프로그램에서 제어문은 명령어의 처리 순서를 변경시키는
    문장입니다. 기본적으로 명령어는 위에서 아래로 차례대로
    처리되는 순차 구조입니다. 하지만 처리 하고자 하는 작업의
    성격에 따라서 순차적인 실행 방법만으로 효율적인 프로그래밍을
    할 수 없습니다. 조건에 따라서 처리하는 명령을 달리하거나
    때로는 특정 명령어를 반복 처리할 필요가 있습니다.
    이처럼 명령의 처리 방법을 지정할때 사용하는 것이 제어문이다.

  . 명령어는 데이터를 처리하기 위한 일정한 논리적인 흐름으로
    구성됩니다. 이처럼 명령어를 논리적으로 구성하는것을
    Algorithm(알고리즘) 이라고 부릅니다. 즉 알고리즘은
    어떠한 문제를 해결하기 위해 명령어를 일정한 제어 구조로
    모아 놓은 것을 의미합니다. 일반적으로 제어 구조는
    순차, 조건, 반복 이라는 세가지로 구성됨

  . 선택문 : 조건에 따라서 처리하는 명령을 달리하는 문장
		   if-else,  switch 문을 사용
		  
  . 반복문 : 일정한 조건을 만족할때까지 동일한 명령을
		   반복적으로 처리하는 문장
		   while, do~while, for문장 사용

  . 분기문 : 강제로 명령어의 처리 순서를 바꾸는 문장입니다.
		    break, continue, return 문을 사용
